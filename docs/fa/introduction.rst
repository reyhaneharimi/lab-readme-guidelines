Introduction (مقدمه)
=====================

این مستندات به منظور استانداردسازی فرایند نگارش سناریوهای آزمایشگاهی (Lab) و README آن‌ها تهیه شده است.

مطالعه این راهنما به شما کمک می‌کند:
- سناریوها را با کیفیت و ساختار ثابت ارائه دهید،
- تیم منتورینگ، توسعه‌دهنده و دانشجو دچار سردرگمی نشوند،
- و روند نگهداری، بازبینی و توسعه سناریوها ساده‌تر گردد.
